## What to build

An orchestrator that combines the original text, extracted entities, RAG literature summary, Causality score, and Trend alerts into a single cohesive "Clinical Investigative Brief". Generates a downloadable PDF or structured JSON for regulatory submission.

## Acceptance criteria

- [ ] Orchestrator aggregates data from all analysis agents
- [ ] Endpoint generates a formatted PDF or JSON document
- [ ] UI provides a "Download Brief" and "Export to JSON" button
- [ ] Exported document strictly adheres to regulatory format requirements
