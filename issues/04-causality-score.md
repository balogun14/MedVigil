## What to build

An agent/logic module to calculate the Naranjo Causality Score. The UI presents the 10-question scale (with some auto-filled by the AI based on the text). The backend calculates the final score and saves it to the report.

## Acceptance criteria

- [ ] UI displays the Naranjo questionnaire for a specific report
- [ ] Backend auto-fills questions where possible based on NLP extraction
- [ ] API calculates and stores the final score (Definite, Probable, Possible, Doubtful)
- [ ] UI prominently displays the causality score next to the reaction
